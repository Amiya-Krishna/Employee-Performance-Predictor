# 👨‍💼 Employee-Performance-Predictor using Data Analytics

## 📌 Project Overview

This project is a **Machine Learning-based Employee Performance Prediction System** that predicts whether an employee will perform as **High, Medium, or Low performer** based on HR-related features.

It helps organizations make **data-driven HR decisions** such as:
- Promotions
- Training allocation
- Performance evaluation
- Workforce planning

---

## 🎯 Problem Statement

In many organizations, employee performance evaluation is manual and can be:

- Biased  
- Inconsistent  
- Time-consuming  

This project solves this problem by using **Machine Learning models** to predict employee performance based on historical HR data.

## ⚙️ How It Works

1. Employee dataset is created or loaded  
2. Data is cleaned and preprocessed  
3. Important features are selected  
4. Machine learning model is trained  
5. Model predicts performance category  
6. Results are analyzed for HR decisions
