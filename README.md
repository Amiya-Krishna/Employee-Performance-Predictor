# 👨‍💼 Employee Performance Predictor using Data Analytics

## 📌 Project Overview

This project uses Machine Learning and Data Analytics to predict employee performance based on HR-related features such as experience, salary, department, training hours, attendance, and past performance.

The goal is to help HR teams identify:
- High-performing employees ⭐
- Medium performers ⚖️
- Low-performing employees ⚠️

This enables better decisions in:
- Promotions
- Training & development
- Workforce planning
- Performance improvement programs

---

## 🎯 Problem Statement

Companies often struggle to objectively evaluate employee performance. Manual appraisal systems can be biased or inconsistent.

👉 This project solves this by using ML to predict performance categories based on historical employee data.

---

## 💡 Business Value

✔ Improves HR decision-making  
✔ Reduces bias in performance reviews  
✔ Identifies employees needing training  
✔ Helps in promotion planning  
✔ Increases workforce productivity
